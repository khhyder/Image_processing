Git at a glance -commands, step by step

	1. Create a remote repo at GitHub
	2. Create a folder at your local machine 
	3. Open that folder on your local machine, right click and select "Git Bash Here"
	4. A command prompt will be pop up. Write "$git init" on command line. Your local repo will be initialize and you could see a .git hidden folder inside your project folder.
	5. Now you can add your local repo with remote repo by writing following command                                                   
	$git remote add origin "http<<your git repo clone link>>" [incase you remove origin try following  command
	$git remote rm origin
	6. To clone/Pull files from remote repo to local repo
$git pull origin master
